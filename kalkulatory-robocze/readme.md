# Kalkulatory robocze

Ten katalog przechowuje wcześniejsze artefakty rapid prototypowania.

## Jak traktować ten folder

- to nie jest source-of-truth dla produktu,
- to nie jest docelowa implementacja,
- to nie jest kanoniczna dokumentacja repo.

Folder służy tylko do:

- porównania wcześniejszych kierunków,
- wyciągania pojedynczych inspiracji UX/UI,
- pokazania przebiegu eksperymentów prototypowych.

## Reguła domyślna

W normalnej pracy nad repo ignoruj ten katalog, chyba że zadanie dotyczy świadomie wcześniejszych prototypów lub porównania kierunków.
