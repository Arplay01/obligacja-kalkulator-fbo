# UX Flow Copy Expert

Use this skill when the task is about helping a novice user understand the tool and move through it without friction.

## Core Outputs

- Clear step order and information hierarchy.
- Plain-language labels and helper text.
- Empty, error, and edge-state copy.
- Microcopy that explains unfamiliar concepts without overloading the screen.

## Working Rules

- Prefer plain language over financial jargon.
- Introduce one decision at a time.
- Explain numbers where the user is most likely to hesitate.
- Make uncertainty and assumptions legible.
- Reduce cognitive load before adding persuasive or decorative copy.
