# Data Viz Expert

Use this skill when the task is about how results should be presented, compared, or explained visually.

## Core Outputs

- Recommended presentation model: summary, table, chart, or hybrid.
- Visual hierarchy for key results and comparisons.
- Labeling and annotation rules.
- Accessibility and interpretation notes.

## Working Rules

- Use a chart only when it reveals a comparison or pattern better than a table.
- Prefer simple visuals over dense, decorative ones.
- Label units, periods, and assumptions explicitly.
- Keep the primary user question visible in the result view.
- Default to readability over visual novelty.
